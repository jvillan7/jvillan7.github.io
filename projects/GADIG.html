GADIG's Website



Synopsis

This website is used to display information on my team: The Game Analysis and Design Interest Group (GADIG). As the current (Fall 2020 - Spring 2021) webmaster, I was tasked with maintaining and updating the website as necessary. This meant updating the roster of current officers as well as improving the website where I see fit.


More Info

Platforms: Desktop; Tablet; Mobile
Software Used: Adobe Dreamweaver; Notepad when only minor changes were needed
Languages: HTML; Javascript; CSS; PHP
Source Control: Filezilla was used to pull and push changes onto a server hosted by Geroge Mason University.
Group: Myself; this was a personal task.


What I programmed and implemented

- The officers of the Meet the Team webpage.
- Installing a Google Calendar plugin




My Thought Processes


Adding Plugins

When I was applying for the role of Webmaster, I was asked how I could improve the website. I chose to make the website more of a centralized hub of information, compared to what it currently was. While people may be interested in what GADIG does, they may not know how to join.

Because of this, I chose to add plugins for Google Calendar to the website. People could then be up to date with what we are currently working on and how to better contact us.




A Change of Officers

With every new school year comes a new set of officers. For this upcoming semester, I communicated with each of the officers, gathering info about themselves and how they became intersted in GADIG.

I then added all of their information onto the website. This was done by implementing a series of 2D arrays via javascript.


What I learned

I learned a bit about how javascript and php work with css to build a website. I also learned a lot about how implementing webiste plugins and how they can make software more interactive for the user.
